GenDoc.rplot — Method
rplot(filename; path_prefix=nothing, uri_prefix=nothing, kwargs...)::String

Wrapper around the ggsave function from Tidyverse. Keyword arguments kwargs are passed to ggsave. Returns a string pointing to the image generated by ggsave. The image is created at path_prefix / filename. To avoid constantly passing path_prefix and url_prefix, consider creating a partial function.
